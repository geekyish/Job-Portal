# Job-Portal
Created job portal for candidates as well as companies using HTML5, CSS3, Javascript, PHP, MySQL

1) Following is the "Register" page, where one can create an account on portal.

![Screenshot (85)](https://user-images.githubusercontent.com/94891505/228729873-00d9c196-e084-4b71-b8f0-9c04ddfa4df1.png)

2) After registration, "Login" page redirects you to admin dashboard.

![Screenshot (84)](https://user-images.githubusercontent.com/94891505/228730016-a47d7abe-b38c-4a9f-a379-2dd44e391747.png)

3) The Admin dashboard enables companies to post the job according to their requirements.

![Screenshot (88)](https://user-images.githubusercontent.com/94891505/228730193-a2cfd35e-7f10-4ad1-b2ee-a33754d0be8a.png)

4) Under the "Post Job" button, employer can add the details of job openings.

![Screenshot (87)](https://user-images.githubusercontent.com/94891505/228730294-95dbf89b-4e64-4277-95e9-6c342bac32c8.png)

5) The job openings can then be seen on the main careers page.

![Screenshot (90)](https://user-images.githubusercontent.com/94891505/228730461-05765fa6-cf06-425c-86f9-c5b9d834e3d8.png)

6) Candidates can apply for any particular job by clicking on "Apply".

![Screenshot (98)](https://user-images.githubusercontent.com/94891505/228730568-ce7d4cee-4b45-466d-b151-491bef8635a9.png)

7) After successfull application, the employer can see the details of candidates in the dashboard.

![Screenshot (89)](https://user-images.githubusercontent.com/94891505/228730732-e0dd5cdc-4e1b-48b2-9b5d-ccd3daa83ea2.png)

8) All the data of user admin, candidates and jobs is stored in the database.
